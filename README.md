# 🌏 Isekai MajinGo

**Isekai MajinGo** 是一款結合「異世界轉生世界觀」與「日文學習」的語言學習工具，透過擷取 YouTube 影片字幕，將真實語境轉換為可學習的語言素材，打造沉浸式的學習體驗。

使用者在觀看 YouTube 影片時，系統可自動取得字幕內容，並將每一句話拆解為可學習的語言單位，包括單字、文法結構與語意解析，讓學習者在「真實語境」中提升日文能力。

在 Isekai MajinGo 的世界中，語言不只是知識，而是一種「魔法能力」。每一句字幕都是一段詠唱，每一次理解都是一次轉生進化。

---

# ⚔️ 核心理念

* 🌐 **從真實內容學習**：直接使用 YouTube 字幕作為語料來源
* 📚 **語句拆解學習**：將句子分解為單字與文法結構
* 🧙 **異世界學習世界觀**：將語言能力視為角色成長與技能提升
* 🔁 **重複輸入強化記憶**：透過影像 + 字幕 + 詠唱式學習強化記憶

---

# 🧠 功能概念

* YouTube 字幕擷取與整理
* 日文句子逐句分析（單字 / 文法 / 句型）
* 語言難度分級（N5～N1）
* 重播式跟讀（Shadowing）學習模式
* 單字收藏與技能樹成長系統

---

# 🔥 世界觀設定

在「Isekai MajinGo」的世界中，學習語言等同於獲得新技能。
使用者作為「語言轉生者」，透過吸收異世界（YouTube影片）的語言能量，不斷提升自身的「Kotoba Level」。

---

# 🚀 目標

打造一個讓「看影片＝學語言」自然發生的學習系統，讓使用者不再只是被動學習，而是主動在內容中進行語言成長。
